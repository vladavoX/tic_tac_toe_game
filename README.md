# Project 03: Tic Tac Toe Game

### This is a Tic Tac Toe game, play it with your friend!

1. Clone this repo
2. enter directory `cd tic_tac_toe_game`
3. run `npm install`
4. run `npm run dev`

When done setting up the project, challenge your friend. <br>
